# نظام إدارة المنزل (House Management System)

نظام متكامل لإدارة المنزل مع نظام منيو، طلبات، تقييمات، إنترنت، ومدفوعات.

## المميزات

- 🍽️ **إدارة المنتجات**: إضافة، تعديل، حذف منتجات مع رفع الصور
- 📦 **إدارة الطلبات**: متابعة حالة الطلبات (Pending, Preparing, Ready, Delivered, Cancelled)
- 💳 **إدارة المدفوعات**: مراجعة المدفوعات، تغيير الحالة، عرض سكرين شوت التحويل
- 🌐 **إدارة الإنترنت**: أكواد إنترنت لمدة 20 دقيقة بسعر 20 جنيه
- ⭐ **التقييمات**: عرض وتقييم الخدمة مع متوسط التقييم
- 📊 **لوحة تحكم**: إحصائيات شاملة عن كل شيء
- 🔐 **مصادقة JWT**: حماية لوحة التحكم
- 🎨 **تصميم حديث**: Glassmorphism، Dark Mode، Responsive

## التقنيات المستخدمة

- **Backend**: Node.js, Express.js
- **Database**: Firebase Firestore
- **Storage**: Cloudinary
- **Authentication**: JWT
- **Hosting**: Vercel

## متطلبات التشغيل

- Node.js v14+
- حساب Firebase (Firestore)
- حساب Cloudinary
- حساب Vercel (للنشر)
